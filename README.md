# Timeshift Panel Indicator
## Svensk beskrivning
Panelprogram som visar Timeshift-status med ikoner.

## Funktioner
- Visar Timeshift-status i systempanelen
- Orange varningsikon när inaktiverat
- Högerklick för meny med alternativ
- Desktop-notiser vid statusändring
- Startar automatiskt vid inloggning

## Installation
1. Ladda ner eller kopiera filerna
2. Kör: ./install.sh
3. Ikonen visas i panelen

## Användning
- **Vänsterklick**: Visa status i tooltip
- **Högerklick**: Öppna meny med alternativ
- **Ikoner**: Grön=aktiverad, Orange=inaktiverad

## Säkerhet
- Öppen källkod
- Inga nätverksanslutningar
- Endast lokal konfigurationsläsning
